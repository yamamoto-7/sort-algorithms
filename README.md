# 1. Merge Sort（マージソート）

## アルゴリズム概要

分割統治法（Divide and Conquer）を用いるソートアルゴリズム。

1. 配列を中央で2分割
2. それぞれを再帰的にソート
3. ソート済み配列同士をマージ

## 計算量

- 平均計算量: O(n log n)
- 最悪計算量: O(n log n)

## 良かった点

- 最悪計算量が常に O(n log n) で安定している
- 安定ソートである（同値要素の相対順序が保たれる）
- アルゴリズム構造が明確で理解しやすい

---

# 2. Quick Sort（クイックソート）

## アルゴリズム概要

1. ピボットを1つ選択
2. ピボットより小さい要素と大きい要素に分割
3. 左右を再帰的にソート
4. 結合

## 計算量

- 平均計算量: O(n log n)
- 最悪計算量: O(n²)

## 良かった点

- 平均的に非常に高速
- 実務でも広く利用されている代表的なアルゴリズム
- 実装がシンプルなため、理解しやすい

---

## 比較

- Merge Sort は最悪計算量が保証されており安定性が強み
- Quick Sort は平均的な処理速度が速く実用性が高い
